Set up working environment

1:Create and download the X.509 certificate and private key on AWS

2:use the scp command to upload the key and certificate to the stevens linux lab in the ~/.ec2 folder.And then use chmod to change the permission that only the me can view it.

3:add following to the ~/.bashrc

export EC2_PRIVATE_KEY=~/.ec2/pk-XXXXXXXXXXXX.pem

export EC2_CERT=~/.ec2/cert-XXXXXXXXXXXX.pem

4:use the source ~/.bashrc to affect the change.

5:also use the chmod to change the bashrc permission so that it will not been viewed by others.

Create an instance(OmniOS)


1:use ec2-describe-images -x all | grep OmniOS to find the ami-id which is ami-505ce739

2:use ec2-create-keypair cs615key to create a keypair for further ssh usage.Copy the output to the cs615key.pem

3:use chmod 400 cs615key.pem to change the permission in order to keep the safety.


4:use ec2-create-group cs615key -d "cs615 hw1 for OmniOS" to create the security group,note the sg-XXXXXXX

5:ec2-authorize cs615key -P tcp -p 22 enable the ssh port can assess it.

6:ec2-run-instances ami-505ce739 -k cs615key -g sg-XXXXXX -t t1.micro to start the instance

7:ec2-describe-instances to know the IP address

8:ssh -i ~/.ssh/cs615key.pem root@XXXXXXXXXX(XXXXXXXXX is the ip address)

9:use exit to close the ssh connection

10:use ec2-describe-instances can know the instance is like i-XXXXXX

11:use ec2-terminate-instances i-XXXXXX can terminate the instance
